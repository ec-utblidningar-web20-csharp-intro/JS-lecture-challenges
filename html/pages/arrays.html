Introduktion

Hela denna sidan går ut på att fördjupa din kunskap i arrayhantering som är ett ämne som är vitt relevant inom programmering.

I grund och botten är det så att allt man vill göra med arrayer går att lösa med hjälp av en for-loop. Problemet är dock att det oftast blir mer kod än vad man hade önskat sig och koden man skriver blir mindre lättläst och på så sätt själva programmet mer svårläst och svårhanterligt.

Därför är tanken med denna sidan att jag går över ett par vanliga hjälpmetoder som finns i alla språk nästan. Du får både pröva på att lösa problemet med en for-loop eller en for-of-loop men också se hur man kan göra samma sak på ett par rader med hjälp av en arrow funktion.

Ett par vanliga array metoder

  • filter
  • map
  • reduce
  • sort

Fråga 3

Börja med att skapa en array med dessa nummer 84, 98, 48, 98, 28, 32, 56, 71, 98, 29.
Använd filter för att filtrera bort alla ojämna tal och skapa en funktion som tar emot ett tal och kollar om det är jämt eller inte.
Funktionen returnerar true om numret är jämnt och annars false.
Skriv ut resultatet. 


const myArray = [84, 75, 48, 5, 28,
                32, 56, 71, 98, 29];
console.log(myArray.filter(isEven));

function isEven(number) {
  if (number % 2 === 0) {
    return true;
  } else {
    return false;
  }
}

Fråga 11

Skapa en array med alla nummer 1-50. Lättast är att skapa en loop som lägger in alla talen så slipper du göra det manuellt.
Skapa en funktion som heter fizzBuzz som tar emot en sträng och ett tal och lägger ihop dessa. Men om talet är jämnt delbar med 3 ska talet bytas ut mot Buzz, om talet är jämnt delbar med 5 ska talet bytas ut mot Fizz och om talet är jämnt delbart med både 3 och 5 ska talet bvytas ut mot Fizz Buzz. Använd sen array.reduce och fizzBuzz för att skriva ut hela arrayen med nummer. 


const myArray = [];
for (let i = 1; i <= 50; i++) {
  myArray.push(i);
}
function fizzBuzz(accumulator, curr) {
  if (curr % 15 === 0) return `${accumulator} Fizz Buzz `;
  if (curr % 5 === 0) return `${accumulator} Fizz `;
  if (curr % 3 === 0) return `${accumulator} Buzz `;
  return `${accumulator}${curr}`;
}
console.log(myArray.reduce(fizzBuzz));

Fråga 14

Gör likadant som på förra uppgiften men byt ut nummerna i arrayen mot strängar. Ändra på din funktion så att den jämför strängar istället för nummer och returnerar.

  • -1 om a är mindre än b
  • 0 om a är lika med b
  • 1 om a är större än b
TIPS. Man kan använda dessa operatorer på strängar <,> och ===. 


const myArray = ['ant', 'bee', 'cat', 'dog', 'elephant'];
function compare(a, b) {
  if (a < b) return -1
  if (a === b) return 0
  if (a > b) return 1
}
console.log(myArray.sort(compare));
