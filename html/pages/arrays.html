Använd filter för att filtrera bort alla ojämna tal och skapa en funktion som tar emot ett tal och kollar om det är jämt eller inte.
Funktionen returnerar true om numret är jämnt och annars false.
Skriv ut resultatet. 


const myArray = [84, 75, 48, 5, 28,
                32, 56, 71, 98, 29];
console.log(myArray.filter(isEven));

function isEven(number) {
  if (number % 2 === 0) {
    return true;
  } else {
    return false;
  }
}

Fråga 7

Skapa en array som heter names och som innehåller några olika namn.
Skapa en funktion som heter reverseName som tar emot ett namn och returnerar namnet i omvänd ordning (oskar blir rakso).
Använda map och reverseName för att ändra ordning på alla namn i names.
TIPS. array.reverse() kan användas för att vända på en array. 


const names = ['Oskar', 'Anders', 'Anna'];
function reverseName(name) {
  return name.split('').reverse().join('');
}
console.log(names.map(reverseName));
